## What to build

Add Cormorant Garamond and IBM Plex Mono to `index.html` via Google Fonts. Expose them as `--serif` and `--mono` CSS variables in `:root` inside `index.css`. Add two layout utilities: `.col-grid` (72 px label | 1 px rule | 1fr content, 16 px gap) and `.top-strip` (flex row, space-between, 16 px 28 px padding). These primitives are prerequisites for every subsequent screen slice.

## Acceptance criteria

- [x] Google Fonts link in `index.html` loads Cormorant Garamond (weights 300, 400) and IBM Plex Mono (weights 400, 500)
- [x] `:root` in `index.css` defines `--serif` and `--mono` pointing to those families
- [x] `.col-grid` utility produces `display: grid; grid-template-columns: 72px 1px 1fr; gap: 16px`
- [x] `.top-strip` utility produces `display: flex; justify-content: space-between; align-items: center; padding: 16px 28px`
- [x] No existing styles are broken by the additions
- [x] Both fonts visibly render in a browser smoke-test

## Tests

No component logic to unit-test. Manual browser smoke-test: open the app and confirm both typefaces load without FOUT. No snapshot tests.
