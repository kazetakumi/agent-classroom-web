## What to build

Rewrite `WelcomeScreen.tsx` to the B-Break structure: column-grid Sage turns showing a personalised greeting (using `localStorage.userName`) and session summary (topic, question count, level, estimated duration), a flex spacer, a "Your move" break section with a single prominent "Begin" button, and an italic `.b-redirect` redirect input ("Or tell Sage something different…") below the button. No Sage strip on this screen.

## Acceptance criteria

- [ ] Sage greeting addresses the user by the stored name from `localStorage.userName`
- [ ] Session summary shows topic, question count, level, and estimated duration
- [ ] "Begin" button is present and triggers session start navigation
- [ ] Italic redirect input is visible below "Begin" button
- [ ] No `SageTrigger` component rendered on this screen
- [ ] Layout uses `.col-grid` for Sage turns and `--serif` / `--mono` fonts correctly

## Tests

- Renders stored name in greeting text
- "Begin" button click calls the `onBegin` prop
- Redirect input is present in the DOM
