## What to build

Create `SageSheet.tsx` — a full-screen overlay that slides up from below when the Sage strip is tapped. Pill handle at top (36×4 px, `#E0E0E0`) dismisses on tap or drag-down. Column-grid layout for Sage turns and a YOU input row. Accepts a `quickActions: string[]` prop that renders contextual action chips (e.g. "give you a hint", "skip this question") — selecting one fades the others. Slide-up animation via CSS transition or react-spring.

After migration, delete `CompanionSheet.tsx`.

Props: `quickActions: string[]`, `onDismiss: () => void`, `onQuickAction: (action: string) => void`.

## Acceptance criteria

- [ ] Overlay slides up from below with a smooth CSS or spring animation
- [ ] Pill handle (36×4 px, `#E0E0E0`) is visible at the top
- [ ] Tapping the pill handle calls `onDismiss`
- [ ] `quickActions` array renders as tappable chips inside the sheet
- [ ] Selecting a quick action fades the non-selected chips
- [ ] Column grid is applied to Sage turn content
- [ ] YOU input row is present for free-text entry
- [ ] `CompanionSheet.tsx` is deleted after all call sites are migrated

## Tests

- Renders quick actions list from `quickActions` prop
- Selecting an action fades the other chips (check class or opacity)
- `onDismiss` is called when pill handle is clicked
- Text input submit triggers the correct handler
