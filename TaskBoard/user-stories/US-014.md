## What to build

Rewrite `QuestionCard.tsx` to the Reading Room aesthetic. No column grid in the question body — full-width reading focus. Subject label in IBM Plex Mono 9 px uppercase. Question text in Cormorant Garamond 21 px / weight 300. Each answer option shows a letter (IBM Plex Mono 9 px `#CFCFCF`) + option text (Cormorant 18 px `#888`). On selection: chosen option letter + text → `#111`; all other options → `opacity: 0.3`. `SageTrigger` with `showNav={true}` anchored at the bottom.

## Acceptance criteria

- [ ] Subject label renders in IBM Plex Mono uppercase
- [ ] Question text renders in Cormorant Garamond at specified size/weight
- [ ] Each option shows letter prefix + option text
- [ ] Selecting an option sets its letter + text to `#111` (full ink)
- [ ] Unselected options after a selection fade to `opacity: 0.3`
- [ ] `SageTrigger` with `showNav={true}` renders at the bottom of the card
- [ ] Component is scoped under `.question-card` CSS namespace

## Tests

- Selecting an option marks it as selected (class or aria attribute)
- Unselected siblings drop to `opacity: 0.3` after a selection
- `SageTrigger` is present in rendered output with `showNav={true}`
